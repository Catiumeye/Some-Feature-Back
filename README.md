## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# to start docker
$ pnpm compose:up

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## To shutdown docker
```bash  
$ pnpm compose:down
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
